# Claude 실무 교육 — 커리큘럼 소개 사이트

AI 전문가 정선웅의 Claude 실무 교육 커리큘럼 안내 랜딩페이지.
설치부터 다중 에이전트까지, 3단계(기초 8강 · 정규 4주 · 심화·특강 8강) 커리큘럼과 오프라인 출강 문의를 담았습니다.

## 구성
- `index.html` — 단일 페이지 (인라인 CSS/JS, 외부 의존성 없음)
- `_images/` — 일러스트·로고·OG 이미지

## 로컬에서 보기
`index.html` 을 브라우저로 열면 됩니다.
