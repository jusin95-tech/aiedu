LESSON 01

에이전틱 AI 시대, Claude로 시작하기

AI를 한 번 쓰고 버리는 검색 도구가 아니라 "매일 같이 일하는 동료"로 보는 관점 전환에서 출발합니다. 계정·환경을 처음부터 함께 세팅합니다.

  • 일반 챗봇과 "에이전틱 AI"의 결정적 차이
  • Claude 계정·플랜·작업 환경 첫 세팅
  • 앞으로 4주를 관통하는 "신입 동료" 프레임 이해
📝
LESSON 02

보고서·이메일·회의록 초단축

가장 많은 시간을 잡아먹는 일상 문서 업무를 몇 배 빠르게. 초안 생성부터 회사 톤 맞추기, 회의록 구조화까지 실습합니다.

  • 보고서·이메일 초안을 한 번에 뽑고 다듬는 법
  • 우리 조직의 톤·형식에 맞추는 요령
  • 회의 메모를 결정사항·액션아이템 구조로 정리
✨
LESSON 03

Artifacts — 결과물 즉시 생성

대화 옆 작업창(Artifacts)에서 문서·간단한 앱·시각 자료를 실시간으로 만들고 바로 고칩니다. 코드를 몰라도 결과물이 눈앞에 나옵니다.

  • Artifacts가 일반 답변과 무엇이 다른가
  • 대화만으로 계산기·랜딩페이지 같은 결과물 생성
  • 결과물을 반복 편집하며 완성도 높이기
🗂️
LESSON 04

Projects — 코워크·팀 협업

맥락(지식·지침)을 기억하는 작업공간으로 매번 배경 설명을 반복하지 않게 만듭니다. 반복 업무를 하나의 시스템으로 바꾸는 첫 단계입니다.

  • 프로젝트 "지식 베이스"와 "지침"의 역할
  • 같은 맥락을 팀과 공유하는 협업 방식
  • 심화 트랙 "프로젝트 실전 활용 4가지"로 이어지는 토대
🔗
LESSON 05

커넥터·스킬·스케줄링 자동화

외부 도구를 연결하고, 반복 작업을 능력으로 묶고, 정해진 시간에 자동 실행시키는 자동화의 기본기를 맛봅니다.

  • 커넥터로 외부 앱·데이터에 연결하기
  • 반복 지시를 "스킬"로 묶는다는 개념 맛보기
  • 정기 작업을 스케줄로 자동화
📊
LESSON 06

데이터 분석·시각화

표·CSV를 올리면 Claude가 직접 분석하고 차트로 보여줍니다. 숫자만 보던 데이터에서 "무엇을 해야 하는지"까지 끌어냅니다.

  • 파일 업로드 → 자동 분석·요약
  • 단순 집계를 넘어 인사이트 도출
  • 결과를 차트·표로 시각화
💻
LESSON 07

Claude Code 입문 — 설치부터 웹까지강의안 완비

터미널에서 돌아가는 AI 코딩 에이전트의 첫걸음. 검정 화면 두려움을 없애고, 설치·오류 해결부터 프롬프트 한 줄로 웹을 만드는 워크플로까지 다룹니다.

  • 플랜 선택(프로 이상)과 터미널·CLI 기본 개념
  • 네이티브 설치 + 대표 오류 2가지 해결(멈춤 / command not found)
  • 토큰 절약(Sonnet 설정)과 탐색→계획→구현 워크플로
🛠️
LESSON 08

Claude Code 실전 — 업무 도구 만들기

입문을 넘어, 내 업무에 딱 맞는 자동화 도구를 직접 만듭니다. 반복 업무 하나를 골라 나만의 작은 프로그램으로 완성합니다.

  • 내 반복 업무를 자동화 도구로 설계
  • 플랜 모드로 안전하게 만들고 검증
  • 완성한 도구를 실제 업무에 적용
